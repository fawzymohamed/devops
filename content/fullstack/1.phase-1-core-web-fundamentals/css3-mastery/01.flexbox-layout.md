# Flexbox Layout

Before Flexbox, creating flexible layouts in CSS required complex float hacks, positioning tricks, and lots of frustration. Flexbox changed everything. It gives you a clean, powerful way to arrange items in a container, distribute space, and align elements.

In this lesson, you'll learn the Flexbox model and master the core properties that make modern layouts easy.

## What Is Flexbox?

**Flexbox** (Flexible Box Layout) is a CSS layout model designed for arranging items in one dimension - either a row or a column. It makes common layout tasks simple:

- Center elements vertically and horizontally
- Distribute space evenly between items
- Create responsive card grids
- Build navigation menus
- Align content flexibly

Flexbox works with two types of elements:
1. **Flex container** - the parent element with `display: flex`
2. **Flex items** - the direct children of the flex container

## The Flexbox Axes

Understanding Flexbox requires knowing about its two axes:

| Axis | Description | Controlled By |
|------|-------------|---------------|
| **Main Axis** | Primary direction (default: horizontal) | `flex-direction` |
| **Cross Axis** | Perpendicular to main axis (default: vertical) | Always perpendicular to main |

```css
/* Default: main axis is horizontal, cross axis is vertical */
.container {
  display: flex;
  flex-direction: row; /* This is the default */
}

/* Change to vertical: main axis is vertical, cross axis is horizontal */
.container {
  display: flex;
  flex-direction: column;
}
```

## Creating a Flex Container

To use Flexbox, set `display: flex` on the container:

```css
.container {
  display: flex;
}
```

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>
```

Immediately, the items arrange horizontally in a row. This is just the start.

## Core Flexbox Properties

### 1. justify-content - Main Axis Alignment

`justify-content` controls how items are distributed along the **main axis** (horizontal by default).

```css
.container {
  display: flex;
  justify-content: flex-start; /* Default - items at the start */
}
```

**Common Values:**

| Value | Effect | Use Case |
|-------|--------|----------|
| `flex-start` | Items at the start (left) | Default behavior |
| `flex-end` | Items at the end (right) | Right-aligned menu |
| `center` | Items centered | Centered button group |
| `space-between` | Space between items | Navigation menu |
| `space-around` | Equal space around items | Icon toolbar |
| `space-evenly` | Equal space including edges | Card layout |

**Example - Navigation Menu:**

```html
<nav class="navbar">
  <a href="/">Home</a>
  <a href="/about">About</a>
  <a href="/contact">Contact</a>
</nav>
```

```css
.navbar {
  display: flex;
  justify-content: space-between;
  padding: 1rem;
  background: #333;
}

.navbar a {
  color: white;
  text-decoration: none;
}
```

### 2. align-items - Cross Axis Alignment

`align-items` controls how items are aligned along the **cross axis** (vertical by default).

```css
.container {
  display: flex;
  align-items: center; /* Vertically center items */
}
```

**Common Values:**

| Value | Effect | Use Case |
|-------|--------|----------|
| `stretch` | Items fill container height (default) | Full-height cards |
| `flex-start` | Items at top | Top-aligned content |
| `flex-end` | Items at bottom | Footer alignment |
| `center` | Items centered vertically | Vertically centered content |
| `baseline` | Items aligned by text baseline | Text alignment |

**Example - Perfect Centering:**

```css
.hero {
  display: flex;
  justify-content: center; /* Horizontal center */
  align-items: center;     /* Vertical center */
  height: 400px;
  background: linear-gradient(to right, #667eea, #764ba2);
  color: white;
}
```

```html
<div class="hero">
  <h1>Welcome to My Site</h1>
</div>
```

### 3. flex-direction - Layout Direction

`flex-direction` determines the main axis direction.

```css
.container {
  display: flex;
  flex-direction: row; /* Default - horizontal */
}
```

**Values:**

| Value | Main Axis | Items Flow |
|-------|-----------|------------|
| `row` | Horizontal → | Left to right (default) |
| `row-reverse` | Horizontal ← | Right to left |
| `column` | Vertical ↓ | Top to bottom |
| `column-reverse` | Vertical ↑ | Bottom to top |

**Example - Vertical Stack:**

```css
.sidebar {
  display: flex;
  flex-direction: column;
  gap: 1rem;
}
```

```html
<aside class="sidebar">
  <div>Widget 1</div>
  <div>Widget 2</div>
  <div>Widget 3</div>
</aside>
```

### 4. flex-wrap - Wrapping Behavior

By default, flex items try to fit on one line. `flex-wrap` allows them to wrap.

```css
.container {
  display: flex;
  flex-wrap: wrap; /* Allow wrapping */
}
```

**Values:**

| Value | Effect |
|-------|--------|
| `nowrap` | All items on one line (default, may overflow) |
| `wrap` | Items wrap to next line if needed |
| `wrap-reverse` | Items wrap to previous line |

**Example - Responsive Card Grid:**

```css
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 1.5rem;
}

.card {
  flex: 0 0 300px; /* Each card is 300px wide */
  padding: 1.5rem;
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}
```

```html
<div class="card-grid">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
  <div class="card">Card 4</div>
</div>
```

### 5. gap - Spacing Between Items

The `gap` property adds space between flex items without using margins.

```css
.container {
  display: flex;
  gap: 1rem; /* 1rem space between all items */
}
```

You can control row and column gaps separately:

```css
.container {
  display: flex;
  flex-wrap: wrap;
  row-gap: 2rem;    /* Vertical space */
  column-gap: 1rem; /* Horizontal space */
}
```

**Example - Button Group:**

```css
.button-group {
  display: flex;
  gap: 0.5rem;
}

.button-group button {
  padding: 0.5rem 1rem;
  border: none;
  background: #3b82f6;
  color: white;
  border-radius: 4px;
  cursor: pointer;
}
```

```html

  display: flex;
  flex-wrap: wrap;
  gap: 2rem;
  padding: 2rem;

  flex: 0 0 calc(33.333% - 2rem); /* 3 columns with gap */
  display: flex;

  padding: 1.5rem;
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);

.product-card img {
  width: 100%;
  border-radius: 4px;
}

.product-card button {
  margin-top: auto; /* Push button to bottom */
  padding: 0.75rem;
  background: #10b981;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

/* Responsive: 2 columns on tablets */
@media (max-width: 768px) {
  .product-card {
    flex: 0 0 calc(50% - 1rem);
  }
}

/* Responsive: 1 column on mobile */
@media (max-width: 480px) {
  .product-card {
    flex: 0 0 100%;
  }
}
```

## Common Flexbox Patterns

**Pattern 1: Holy Grail Layout**

```css
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

header { /* Fixed height */ }

main {
  display: flex;
  flex: 1; /* Grow to fill space */
}

aside { width: 250px; }

article { flex: 1; }

footer { /* Fixed height */ }
```

**Pattern 2: Equal Height Columns**

```css
.columns {
  display: flex;
  gap: 2rem;
}

.column {
  flex: 1; /* All columns equal width */
}
```

**Pattern 3: Sticky Footer**

```css
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

main {
  flex: 1; /* Grows to push footer down */
}

footer {
  /* Stays at bottom */
}
```

## Best Practices

1. **Use Flexbox for one-dimensional layouts**
   - Use Flexbox for rows or columns
   - Use Grid for two-dimensional layouts

2. **Start with mobile-first design**
   - Use `flex-direction: column` by default
   - Switch to `row` on larger screens

3. **Use gap instead of margins**
   - `gap` is cleaner and more maintainable
   - No need for negative margins on containers

4. **Combine with other properties**
   - Flexbox works great with `max-width`, `padding`, etc.
   - Use `flex-wrap` for responsive layouts

5. **Test in different browsers**
   - Flexbox has excellent support but test edge cases
   - Use autoprefixer for older browser support

## Common Mistakes to Avoid

1. **Forgetting display: flex**
   - Flexbox properties only work on flex containers
   - Remember to set `display: flex` on the parent

2. **Confusing justify-content and align-items**
   - `justify-content` is for the main axis
   - `align-items` is for the cross axis
   - The axis direction depends on `flex-direction`

3. **Not using flex-wrap for responsive layouts**
   - Without `flex-wrap: wrap`, items may overflow
   - Always consider wrapping for multi-item layouts

4. **Applying flex properties to the wrong element**
   - Container properties: `justify-content`, `align-items`, `flex-direction`, `flex-wrap`, `gap`
   - Item properties: `flex`, `order`, `align-self`

## Key Takeaways

- **Flexbox** is a one-dimensional layout model for rows or columns
- **display: flex** creates a flex container, making children flex items
- **justify-content** controls spacing along the main axis (horizontal by default)
- **align-items** controls alignment along the cross axis (vertical by default)
- **flex-direction** sets the main axis direction (row, column, row-reverse, column-reverse)
- **flex-wrap** allows items to wrap to multiple lines for responsive layouts
- **gap** adds space between items without margins
- Use Flexbox for navbars, card grids, centering content, and flexible layouts

Flexbox is one of the most important CSS skills for modern web development. Master these properties, and you'll build layouts faster and cleaner than ever before.
