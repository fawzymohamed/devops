# CSS Grid Layout

CSS Grid is the most powerful layout system in CSS. While Flexbox arranges items in one dimension (either rows or columns), Grid lets you control both at once. You can create complex layouts that were once impossible or required messy hacks.

In this lesson, you'll learn how Grid works, master the core properties, and build responsive layouts that adapt to any screen size.

## What Is CSS Grid?

**CSS Grid** is a two-dimensional layout system that divides a container into rows and columns, creating cells where you can place items. Think of it like a spreadsheet for your layout.

Grid works with two types of elements:
1. **Grid container** - the parent element with `display: grid`
2. **Grid items** - the direct children of the grid container

**When to Use Grid vs Flexbox:**

| Use CSS Grid | Use Flexbox |
|--------------|-------------|
| Two-dimensional layouts (rows AND columns) | One-dimensional layouts (rows OR columns) |
| Full page layouts | Navigation bars |
| Complex dashboards | Button groups |
| Image galleries | Card components |
| Form layouts | Vertically centering content |

**Best Practice:** Use Grid for the overall layout structure, then use Flexbox inside grid items for component-level layouts.

## Creating a Grid Container

To use CSS Grid, set `display: grid` on the container:

```css
.container {
  display: grid;
}
```

```html
<div class="container">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
</div>
```

Right now, items stack vertically in one column. You need to define columns and rows.

## Defining Columns and Rows

### grid-template-columns

Use `grid-template-columns` to define column widths:

```css
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px; /* Three 200px columns */
}
```

This creates three columns, each 200 pixels wide.

**Example - Three Column Layout:**

```css
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  gap: 1rem;
}
```

```html
<div class="container">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
  <div>Column 4</div>
  <div>Column 5</div>
  <div>Column 6</div>
</div>
```

Items automatically flow into the grid, wrapping to new rows when needed.

### grid-template-rows

Use `grid-template-rows` to define row heights:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 100px 200px; /* First row 100px, second row 200px */
}
```

If you don't specify rows, they auto-size to fit content.

## The Powerful fr Unit

The **fr** unit (fraction) represents a fraction of available space. It's one of Grid's most useful features.

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr; /* Middle column is twice as wide */
}
```

**How fr Works:**

1. Grid calculates total fractions (1 + 2 + 1 = 4)
2. Divides available space by 4
3. First column gets 1/4, middle gets 2/4 (half), last gets 1/4

**Example - Sidebar Layout:**

```css
.page {
  display: grid;
  grid-template-columns: 250px 1fr; /* Fixed sidebar, flexible content */
  gap: 2rem;
  min-height: 100vh;
}
```

```html
<div class="page">
  <aside class="sidebar">
    Sidebar content
  </aside>
  <main class="content">
    Main content
  </main>
</div>
```

## The repeat() Function

Instead of writing the same value multiple times, use `repeat()`:

```css
/* Instead of: grid-template-columns: 1fr 1fr 1fr 1fr; */
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* Four equal columns */
}
```

**Syntax:** `repeat(count, size)`

**Examples:**

```css
/* Four equal columns */
grid-template-columns: repeat(4, 1fr);

/* Three columns: 100px, 1fr, 100px */
grid-template-columns: 100px repeat(1, 1fr) 100px;

/* Six columns with mixed sizes */
grid-template-columns: repeat(3, 1fr 2fr);
/* Expands to: 1fr 2fr 1fr 2fr 1fr 2fr */
```

## gap - Spacing Between Grid Items

The `gap` property adds space between grid rows and columns:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem; /* 1rem space between all cells */
}
```

You can control row and column gaps separately:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  row-gap: 2rem;    /* Vertical space */
  column-gap: 1rem; /* Horizontal space */
}
```

**Old Syntax (still works):**
- `grid-gap` instead of `gap`
- `grid-row-gap` instead of `row-gap`
- `grid-column-gap` instead of `column-gap`

## Responsive Grids with auto-fit and auto-fill

These keywords create responsive grids that adapt to screen size without media queries.

### auto-fill

`auto-fill` creates as many columns as fit, keeping empty tracks:

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 1rem;
}
```

**How it works:**
1. Each column is at least 250px wide
2. Columns grow up to 1fr of space
3. Creates as many columns as fit, even if some are empty

### auto-fit

`auto-fit` is like `auto-fill`, but collapses empty tracks:

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}
```

**Difference from auto-fill:**
- **auto-fill**: Keeps empty tracks, items don't stretch beyond minmax size
- **auto-fit**: Collapses empty tracks, items stretch to fill space

**Visual Comparison:**

| Items | auto-fill | auto-fit |
|-------|-----------|----------|
| 3 items, space for 5 | 3 items + 2 empty tracks | 3 items stretched to fill |
| 6 items, space for 6 | 6 items exactly | 6 items exactly |

**Example - Responsive Card Grid:**

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
}

.card {
  padding: 1.5rem;
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}
```

```html
<div class="card-grid">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
  <div class="card">Card 4</div>
</div>
```

This grid automatically adjusts:
- **Large screens**: Multiple columns
- **Medium screens**: Fewer columns
- **Small screens**: Single column

No media queries needed!

## minmax() Function

`minmax(min, max)` sets minimum and maximum sizes:

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, minmax(200px, 1fr));
}
```

Columns will be:
- **Minimum**: 200px
- **Maximum**: 1fr (equal share of remaining space)

**Common Patterns:**

```css
/* Columns between 150px and 300px */
grid-template-columns: repeat(auto-fit, minmax(150px, 300px));

/* Columns at least 250px, grow to fill */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* Fixed sidebar, flexible content (min 300px) */
grid-template-columns: 250px minmax(300px, 1fr);
```

## Common Grid Properties Reference

**Container Properties:**

| Property | Purpose | Example |
|----------|---------|---------|
| `display: grid` | Creates grid container | `display: grid` |
| `grid-template-columns` | Defines column sizes | `repeat(3, 1fr)` |
| `grid-template-rows` | Defines row sizes | `100px auto 100px` |
| `gap` | Space between items | `1rem` |
| `row-gap` | Vertical space | `2rem` |
| `column-gap` | Horizontal space | `1rem` |
| `justify-items` | Horizontal alignment | `center` |
| `align-items` | Vertical alignment | `center` |

**Item Properties (covered in advanced lessons):**

| Property | Purpose | Example |
|----------|---------|---------|
| `grid-column` | Column span | `1 / 3` |
| `grid-row` | Row span | `1 / 3` |
| `grid-area` | Named area | `header` |

## Best Practices

1. **Use Grid for page layout, Flexbox for components**
   - Grid: Overall structure
   - Flexbox: Inside grid items

2. **Use fr units for flexibility**
   - Prefer `1fr` over percentages
   - Combine fixed and flexible: `250px 1fr`

3. **Use auto-fit/auto-fill for responsive grids**
   - Minimal media queries needed
   - `repeat(auto-fit, minmax(250px, 1fr))`

4. **Set explicit gaps**
   - Use `gap` instead of margins on items
   - Cleaner and more maintainable

5. **Name your grid areas for complex layouts**
   - Makes templates more readable
   - Easier to maintain

## Common Mistakes to Avoid

1. **Forgetting display: grid**
   - Grid properties only work on grid containers
   - Always set `display: grid` first

2. **Confusing auto-fit and auto-fill**
   - Use `auto-fit` when you want items to stretch
   - Use `auto-fill` when you want consistent sizing

3. **Not using minmax() with auto-fit/auto-fill**
   - Always wrap in `minmax(min-width, 1fr)`
   - Without it, columns can collapse too small

4. **Overcomplicating with too many rows/columns**
   - Start simple, add complexity as needed
   - Use implicit grid (auto rows) when possible

## Key Takeaways

- **CSS Grid** is for two-dimensional layouts (rows AND columns)
- **display: grid** creates a grid container
- **grid-template-columns** defines column sizes
- **grid-template-rows** defines row sizes (often auto-sized)
- **fr unit** represents a fraction of available space
- **repeat()** creates multiple columns/rows with one value
- **gap** adds space between grid cells
- **auto-fit** collapses empty tracks, stretching items
- **auto-fill** keeps empty tracks, maintaining sizing
- **minmax()** sets minimum and maximum sizes for responsive grids
- Use Grid for layouts, Flexbox for components

CSS Grid transforms how you build layouts. Master these properties, and you'll create complex, responsive designs with clean, maintainable code.
