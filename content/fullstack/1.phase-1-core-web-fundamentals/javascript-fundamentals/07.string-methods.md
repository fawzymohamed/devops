# String Methods

Strings are everywhere in JavaScript - user input, API responses, file paths, URLs. Knowing how to manipulate strings efficiently is a fundamental skill.

This lesson covers the most useful string methods you'll use daily: extracting parts, searching, replacing, and formatting.

## Strings Are Immutable

Before diving in, remember: **strings are immutable**. String methods don't modify the original string - they return a new one.

```javascript
const original = "Hello";
const upper = original.toUpperCase();

console.log(original);  // "Hello" - unchanged
console.log(upper);     // "HELLO" - new string
```

## Extracting Substrings

### slice(start, end)

Extract a portion of a string from `start` index to `end` (not including `end`):

```javascript
const str = "Hello, World!";

str.slice(0, 5);     // "Hello"
str.slice(7, 12);    // "World"
str.slice(7);        // "World!" - to end if no second argument
```

**Negative indexes** count from the end:

```javascript
const str = "Hello, World!";

str.slice(-6);       // "World!" - last 6 characters
str.slice(-6, -1);   // "World" - last 6, excluding last 1
str.slice(0, -1);    // "Hello, World" - all except last character
```

### substring(start, end)

Similar to `slice()`, but no negative indexes:

```javascript
const str = "Hello, World!";

str.substring(0, 5);   // "Hello"
str.substring(7, 12);  // "World"
str.substring(7);      // "World!"
```

**Difference from slice():** If `start > end`, substring swaps them:

```javascript
str.slice(5, 0);       // "" - empty (no swap)
str.substring(5, 0);   // "Hello" - swapped to (0, 5)
```

**Best practice:** Prefer `slice()` - it's more predictable.

## Splitting and Joining

### split(separator)

Convert a string to an array by splitting at the separator:

```javascript
const sentence = "Hello World JavaScript";
const words = sentence.split(" ");
console.log(words);  // ["Hello", "World", "JavaScript"]

const csv = "apple,banana,orange";
const fruits = csv.split(",");
console.log(fruits);  // ["apple", "banana", "orange"]

// Split into characters
const chars = "Hello".split("");
console.log(chars);  // ["H", "e", "l", "l", "o"]
```

**Limit the number of splits:**

```javascript
const path = "/users/john/documents/file.txt";
const parts = path.split("/", 3);
console.log(parts);  // ["", "users", "john"]
```

### join(separator)

Convert an array back to a string:

```javascript
const words = ["Hello", "World"];

words.join(" ");    // "Hello World"
words.join("-");    // "Hello-World"
words.join("");     // "HelloWorld"
words.join(", ");   // "Hello, World"
```

**Common pattern - split, transform, join:**

```javascript
// Convert "hello-world" to "Hello World"
const slug = "hello-world";
const title = slug
  .split("-")
  .map(word => word.charAt(0).toUpperCase() + word.slice(1))
  .join(" ");

console.log(title);  // "Hello World"
```

## Searching Strings

### includes(searchString)

Check if a string contains a substring (returns boolean):

```javascript
const text = "Hello, World!";

text.includes("World");   // true
text.includes("world");   // false (case-sensitive)
text.includes("xyz");     // false
```

**Start search from a specific position:**

```javascript
text.includes("o", 5);    // true - search from index 5
text.includes("H", 5);    // false - "H" is before index 5
```

### indexOf(searchString)

Find the position of a substring (returns index or -1):

```javascript
const text = "Hello, World!";

text.indexOf("World");    // 7
text.indexOf("o");        // 4 (first occurrence)
text.indexOf("xyz");      // -1 (not found)
```

**Start search from a specific position:**

```javascript
text.indexOf("o", 5);     // 8 - find "o" starting from index 5
```

### lastIndexOf(searchString)

Find the last occurrence:

```javascript
const text = "Hello, World!";

text.lastIndexOf("o");    // 8 (last "o")
text.indexOf("o");        // 4 (first "o")
```

### startsWith(searchString) and endsWith(searchString)

Check the beginning or end of a string:

```javascript
const filename = "document.pdf";

filename.startsWith("doc");       // true
filename.startsWith("Doc");       // false (case-sensitive)
filename.endsWith(".pdf");        // true
filename.endsWith(".txt");        // false
```

**Practical uses:**

```javascript
// Check file types
const isPDF = filename.endsWith(".pdf");
const isImage = filename.endsWith(".jpg") || filename.endsWith(".png");

// Check URL protocols
const url = "https://example.com";
const isSecure = url.startsWith("https://");

// Check prefixes
const id = "user_12345";
const isUserId = id.startsWith("user_");
```

## Replacing Content

### replace(search, replacement)

Replace the **first occurrence** of a substring:

```javascript
const text = "Hello World World";

text.replace("World", "JavaScript");
// "Hello JavaScript World" - only first is replaced
```

### replaceAll(search, replacement)

Replace **all occurrences**:

```javascript
const text = "Hello World World";

text.replaceAll("World", "JavaScript");
// "Hello JavaScript JavaScript"
```

### Using Regular Expressions

For more powerful replacements:

```javascript
// Case-insensitive replacement
const text = "Hello WORLD world";
text.replace(/world/gi, "JavaScript");
// "Hello JavaScript JavaScript"

// Replace multiple patterns
const messy = "  too   many    spaces  ";
messy.replace(/\s+/g, " ").trim();
// "too many spaces"
```

### Dynamic Replacement

Use a function for dynamic replacements:

```javascript
const text = "Hello World";

const result = text.replace(/\w+/g, (word) => {
  return word.toUpperCase();
});

console.log(result);  // "HELLO WORLD"
```

## Case Conversion

### toUpperCase() and toLowerCase()

```javascript
const text = "Hello World";

text.toUpperCase();   // "HELLO WORLD"
text.toLowerCase();   // "hello world"
```

**Common use - case-insensitive comparison:**

```javascript
const input = "YES";
const answer = "yes";

// Case-insensitive comparison
if (input.toLowerCase() === answer.toLowerCase()) {
  console.log("Match!");
}
```

### Capitalizing Words

JavaScript doesn't have a built-in capitalize method, so create your own:

```javascript
// Capitalize first letter
const capitalize = (str) => {
  return str.charAt(0).toUpperCase() + str.slice(1).toLowerCase();
};

capitalize("hello");   // "Hello"
capitalize("HELLO");   // "Hello"

// Capitalize each word (title case)
const titleCase = (str) => {
  return str
    .toLowerCase()
    .split(" ")
    .map(word => word.charAt(0).toUpperCase() + word.slice(1))
    .join(" ");
};

titleCase("hello world");  // "Hello World"
```

## Trimming Whitespace

### trim(), trimStart(), trimEnd()

Remove whitespace from strings:

```javascript
const messy = "   Hello World   ";

messy.trim();       // "Hello World" - both ends
messy.trimStart();  // "Hello World   " - only start (left)
messy.trimEnd();    // "   Hello World" - only end (right)
```

**Common use - cleaning user input:**

```javascript
const userInput = "  john@example.com  ";
const email = userInput.trim().toLowerCase();
console.log(email);  // "john@example.com"
```

## Padding Strings

### padStart(targetLength, padString)

Pad the beginning of a string:

```javascript
const num = "42";

num.padStart(5, "0");   // "00042"
num.padStart(5, " ");   // "   42"
num.padStart(5, ".");   // "...42"
```

### padEnd(targetLength, padString)

Pad the end of a string:

```javascript
const name = "John";

name.padEnd(10, ".");   // "John......"
name.padEnd(10, " ");   // "John      "
```

**Practical uses:**

```javascript
// Format numbers (IDs, codes)
const id = "7";
const formattedId = id.padStart(4, "0");  // "0007"

// Format time
const hours = "9";
const minutes = "5";
const time = `${hours.padStart(2, "0")}:${minutes.padStart(2, "0")}`;
// "09:05"

// Align text in console
const items = ["Apple", "Banana", "Cherry"];
items.forEach(item => {
  console.log(item.padEnd(10, ".") + " $1.00");
});
// Apple..... $1.00
// Banana.... $1.00
// Cherry.... $1.00
```

## Repeating Strings

### repeat(count)

Repeat a string multiple times:

```javascript
const star = "*";

star.repeat(5);     // "*****"
star.repeat(0);     // ""

// Create a divider
const divider = "-".repeat(50);
console.log(divider);  // "--------------------------------------------------"
```

## Getting Characters

### charAt(index) and Bracket Notation

```javascript
const str = "Hello";

str.charAt(0);    // "H"
str.charAt(1);    // "e"
str[0];           // "H" - bracket notation
str[4];           // "o"

// Out of range
str.charAt(10);   // "" - empty string
str[10];          // undefined
```

### at(index)

Modern method that supports negative indexes:

```javascript
const str = "Hello";

str.at(0);     // "H"
str.at(-1);    // "o" - last character
str.at(-2);    // "l" - second to last
```

## Practical Examples

### Example 1: Validate Email Format

```javascript
const isValidEmail = (email) => {
  const trimmed = email.trim().toLowerCase();
  return trimmed.includes("@") &&
         trimmed.includes(".") &&
         trimmed.indexOf("@") > 0 &&
         trimmed.lastIndexOf(".") > trimmed.indexOf("@");
};

isValidEmail("john@example.com");   // true
isValidEmail("invalid-email");       // false
```

### Example 2: Format Phone Number

```javascript
const formatPhone = (phone) => {
  // Remove all non-digits
  const digits = phone.replace(/\D/g, "");

  // Format as (XXX) XXX-XXXX
  if (digits.length === 10) {
    return `(${digits.slice(0, 3)}) ${digits.slice(3, 6)}-${digits.slice(6)}`;
  }
  return phone;
};

formatPhone("1234567890");      // "(123) 456-7890"
formatPhone("123-456-7890");    // "(123) 456-7890"
```

### Example 3: Truncate Long Text

```javascript
const truncate = (text, maxLength = 100) => {
  if (text.length <= maxLength) return text;
  return text.slice(0, maxLength - 3).trim() + "...";
};

truncate("This is a very long text", 15);
// "This is a ve..."
```

### Example 4: Slug Generator

```javascript
const slugify = (text) => {
  return text
    .toLowerCase()
    .trim()
    .replace(/[^\w\s-]/g, "")     // Remove special chars
    .replace(/\s+/g, "-")         // Replace spaces with hyphens
    .replace(/-+/g, "-");         // Remove duplicate hyphens
};

slugify("Hello World! How are you?");
// "hello-world-how-are-you"
```

### Example 5: Parse CSV Line

```javascript
const parseCSVLine = (line) => {
  return line
    .split(",")
    .map(value => value.trim());
};

parseCSVLine("John, Doe, john@example.com");
// ["John", "Doe", "john@example.com"]
```

## Method Chaining

String methods return strings, so you can chain them:

```javascript
const input = "  HELLO WORLD  ";

const result = input
  .trim()
  .toLowerCase()
  .replace("world", "JavaScript")
  .split(" ")
  .map(word => word.charAt(0).toUpperCase() + word.slice(1))
  .join(" ");

console.log(result);  // "Hello Javascript"
```

## Key Takeaways

- Strings are **immutable** - methods return new strings
- **slice()** extracts substrings, supports negative indexes
- **split()** converts string to array, **join()** converts array to string
- **includes()** checks for substring existence (boolean)
- **indexOf()** returns position or -1 if not found
- **startsWith()/endsWith()** check string boundaries
- **replace()** replaces first occurrence, **replaceAll()** replaces all
- **trim()** removes whitespace from both ends
- **padStart()/padEnd()** add characters to reach target length
- **toUpperCase()/toLowerCase()** for case conversion
- **at()** supports negative indexes for accessing characters
- Chain methods for complex string transformations

String manipulation is one of the most common tasks in JavaScript. Master these methods, and you'll handle user input, format data, and process text with ease.
