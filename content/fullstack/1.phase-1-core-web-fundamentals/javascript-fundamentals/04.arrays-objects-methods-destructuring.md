# Arrays & Objects - Methods, Destructuring, and Spread

Arrays and objects are JavaScript's workhorses for storing data. Arrays hold ordered lists (like a shopping list), while objects hold key-value pairs (like a person's profile with name, age, email).

Modern JavaScript gives you powerful tools to work with them: **array methods** for transforming data, **destructuring** for extracting values cleanly, and the **spread operator** for copying and merging.

In this lesson, you'll master these essential techniques and learn when to use each one.

## Arrays - Ordered Lists

Arrays store multiple values in order. Each value has a numbered position (index) starting at 0.

**Creating Arrays:**

```javascript
// Empty array
const emptyArray = [];

// Array with values
const fruits = ["apple", "banana", "orange"];

// Array with different types (not recommended)
const mixed = [1, "hello", true, null];
```

**Accessing Elements:**

```javascript
const colors = ["red", "green", "blue"];

console.log(colors[0]);  // Output: red
console.log(colors[1]);  // Output: green
console.log(colors[2]);  // Output: blue
console.log(colors.length); // Output: 3
```

## Essential Array Methods

JavaScript provides many built-in methods to work with arrays. These methods make data manipulation easy and readable.

### 1. `map()` - Transform Each Element

Creates a new array by applying a function to each element.

```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2);

console.log(doubled); // Output: [2, 4, 6, 8]
console.log(numbers); // Output: [1, 2, 3, 4] (unchanged)
```

**Use Case:** Converting data (e.g., extracting names from user objects)

```javascript
const users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 }
];

const names = users.map(user => user.name);
console.log(names); // Output: ["Alice", "Bob"]
```

### 2. `filter()` - Select Elements

Creates a new array with elements that pass a test.

```javascript
const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter(n => n % 2 === 0);

console.log(evenNumbers); // Output: [2, 4, 6]
```

**Use Case:** Finding specific items

```javascript
const products = [
  { name: "Laptop", price: 1000 },
  { name: "Mouse", price: 25 },
  { name: "Keyboard", price: 75 }
];

const affordable = products.filter(p => p.price < 100);
console.log(affordable);
// Output: [{ name: "Mouse", price: 25 }, { name: "Keyboard", price: 75 }]
```

### 3. `find()` - Get First Match

Returns the first element that passes a test (or `undefined` if none found).

```javascript
const users = [
  { id: 1, name: "Alice" },
  { id: 2, name: "Bob" },
  { id: 3, name: "Charlie" }
];

const user = users.find(u => u.id === 2);
console.log(user); // Output: { id: 2, name: "Bob" }
```

### 4. `reduce()` - Combine Elements

Combines all elements into a single value.

```javascript
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((total, n) => total + n, 0);

console.log(sum); // Output: 10
```

**Breakdown:**
- `total` - accumulator (starts at 0)
- `n` - current element
- `0` - initial value

### 5. Other Useful Methods

```javascript
const fruits = ["apple", "banana", "orange"];

// Add to end
fruits.push("grape");
console.log(fruits); // Output: ["apple", "banana", "orange", "grape"]

// Remove from end
const last = fruits.pop();
console.log(last);   // Output: grape
console.log(fruits); // Output: ["apple", "banana", "orange"]

// Add to beginning
fruits.unshift("mango");
console.log(fruits); // Output: ["mango", "apple", "banana", "orange"]

// Remove from beginning
const first = fruits.shift();
console.log(first);  // Output: mango

// Check if includes value
console.log(fruits.includes("banana")); // Output: true

// Find index
console.log(fruits.indexOf("orange")); // Output: 2
```

## Objects - Key-Value Pairs

Objects store data as key-value pairs. Think of them like a dictionary where you look up values by their keys.

**Creating Objects:**

```javascript
// Empty object
const emptyObj = {};

// Object with properties
const person = {
  name: "Alice",
  age: 25,
  city: "New York"
};
```

**Accessing Properties:**

```javascript
const user = {
  name: "Bob",
  email: "bob@example.com",
  isActive: true
};

// Dot notation
console.log(user.name);  // Output: Bob

// Bracket notation (when key is dynamic or has spaces)
console.log(user["email"]); // Output: bob@example.com

const key = "isActive";
console.log(user[key]); // Output: true
```

**Modifying Objects:**

```javascript
const car = {
  brand: "Toyota",
  year: 2020
};

// Add property
car.color = "red";

// Update property
car.year = 2021;

// Delete property
delete car.color;

console.log(car); // Output: { brand: "Toyota", year: 2021 }
```

## Destructuring - Extract Values Cleanly

Destructuring lets you unpack values from arrays or objects into separate variables. It's cleaner than accessing elements one by one.

### Array Destructuring

Extract values from arrays by position:

```javascript
const colors = ["red", "green", "blue"];

// Old way
const first = colors[0];
const second = colors[1];

// Destructuring way
const [first, second, third] = colors;
console.log(first);  // Output: red
console.log(second); // Output: green
console.log(third);  // Output: blue
```

**Skip Elements:**

```javascript
const numbers = [1, 2, 3, 4, 5];
const [first, , third] = numbers; // Skip second

console.log(first); // Output: 1
console.log(third); // Output: 3
```

**Default Values:**

```javascript
const [a, b, c = 10] = [1, 2];
console.log(c); // Output: 10 (default value)
```

### Object Destructuring

Extract values from objects by key:

```javascript
const user = {
  name: "Alice",
  age: 25,
  city: "Paris"
};

// Old way
const name = user.name;
const age = user.age;

// Destructuring way
const { name, age, city } = user;
console.log(name); // Output: Alice
console.log(age);  // Output: 25
console.log(city); // Output: Paris
```

**Rename Variables:**

```javascript
const user = { name: "Bob", age: 30 };
const { name: userName, age: userAge } = user;

console.log(userName); // Output: Bob
console.log(userAge);  // Output: 30
```

**Default Values:**

```javascript
const user = { name: "Charlie" };
const { name, age = 18 } = user;

console.log(age); // Output: 18 (default value)
```

**Nested Destructuring:**

```javascript
const user = {
  name: "Alice",
  address: {
    city: "London",
    zip: "12345"
  }
};

const { name, address: { city, zip } } = user;
console.log(city); // Output: London
console.log(zip);  // Output: 12345
```

## Spread Operator (...) - Copy and Merge

The spread operator expands arrays and objects. Use it to copy, merge, or pass elements as individual arguments.

### Spread with Arrays

**Copy an Array:**

```javascript
const original = [1, 2, 3];
const copy = [...original];

copy.push(4);

console.log(original); // Output: [1, 2, 3] (unchanged)
console.log(copy);     // Output: [1, 2, 3, 4]
```

**Merge Arrays:**

```javascript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const merged = [...arr1, ...arr2];

console.log(merged); // Output: [1, 2, 3, 4, 5, 6]
```

**Add Elements:**

```javascript
const numbers = [2, 3, 4];
const withEnds = [1, ...numbers, 5];

console.log(withEnds); // Output: [1, 2, 3, 4, 5]
```

### Spread with Objects

**Copy an Object:**

```javascript
const original = { name: "Alice", age: 25 };
const copy = { ...original };

copy.city = "Paris";

console.log(original); // Output: { name: "Alice", age: 25 }
console.log(copy);     // Output: { name: "Alice", age: 25, city: "Paris" }
```

**Merge Objects:**

```javascript
const defaults = { theme: "dark", lang: "en" };
const userPrefs = { lang: "fr", fontSize: 14 };

const settings = { ...defaults, ...userPrefs };
console.log(settings);
// Output: { theme: "dark", lang: "fr", fontSize: 14 }
// Note: userPrefs.lang overwrites defaults.lang
```

**Update Properties:**

```javascript
const user = { name: "Bob", age: 30, city: "NYC" };
const updated = { ...user, age: 31 };

console.log(updated); // Output: { name: "Bob", age: 31, city: "NYC" }
```

## Practical Examples

### Example 1: User List with Methods

```javascript
const users = [
  { id: 1, name: "Alice", active: true },
  { id: 2, name: "Bob", active: false },
  { id: 3, name: "Charlie", active: true }
];

// Get active users
const activeUsers = users.filter(u => u.active);
console.log(activeUsers);
// Output: [{ id: 1, name: "Alice", active: true }, { id: 3, name: "Charlie", active: true }]

// Get just names
const names = users.map(u => u.name);
console.log(names); // Output: ["Alice", "Bob", "Charlie"]

// Find user by id
const user = users.find(u => u.id === 2);
console.log(user); // Output: { id: 2, name: "Bob", active: false }
```

### Example 2: Destructuring in Function Parameters

```javascript
// Extract object properties in function parameters
const printUser = ({ name, age }) => {
  console.log(`${name} is ${age} years old`);
};

const user = { name: "Alice", age: 25, city: "Paris" };
printUser(user); // Output: Alice is 25 years old
```

### Example 3: Combining Arrays and Objects

```javascript
const products = [
  { id: 1, name: "Laptop", price: 1000 },
  { id: 2, name: "Mouse", price: 25 }
];

// Add discount to each product
const discounted = products.map(p => ({
  ...p,
  price: p.price * 0.9
}));

console.log(discounted);
// Output: [{ id: 1, name: "Laptop", price: 900 }, { id: 2, name: "Mouse", price: 22.5 }]
```

## Common Mistakes to Avoid

1. **Mutating arrays with push/pop in functional code**
   - Methods like `map` and `filter` create new arrays
   - Don't mix them with mutating methods like `push`

2. **Confusing spread with rest operator**
   - Spread **expands**: `[...arr]` or `{ ...obj }`
   - Rest **collects**: `function(...args)` or `const [first, ...rest] = arr`

3. **Shallow copy with spread**
   - Spread creates shallow copies
   - Nested objects/arrays are still referenced, not copied

4. **Forgetting return in arrow functions**
   - `arr.map(x => x * 2)` ‚úÖ (implicit return)
   - `arr.map(x => { x * 2 })` ‚ùå (no return, returns undefined)

5. **Using bracket notation when dot notation works**
   - Use `obj.name` for fixed keys
   - Use `obj[key]` for dynamic keys only

## Key Takeaways

- **Array methods** like `map`, `filter`, and `find` make data manipulation clean and functional
- **Destructuring** extracts values from arrays and objects in a concise, readable way
- The **spread operator** copies and merges arrays and objects without mutation
- Arrays hold ordered data, objects hold key-value data - choose based on your needs
- Modern JavaScript favors immutability - create new arrays/objects instead of modifying originals
- Chain methods for powerful transformations: `arr.filter(...).map(...).reduce(...)`

These tools are fundamental to modern JavaScript. You'll use them constantly in frameworks like Vue and React. Practice combining them to build clean, efficient data transformations.
