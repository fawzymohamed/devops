# JavaScript Fundamentals Cheat Sheet

Quick reference for JavaScript variables, data types, functions, arrays, objects, control flow, and ES6+ features.

---

## Variables & Scope

### Declaration Keywords

| Keyword | Scope | Reassign | Redeclare | Hoisted |
|---------|-------|----------|-----------|---------|
| `const` | Block | No | No | TDZ |
| `let` | Block | Yes | No | TDZ |
| `var` | Function | Yes | Yes | Yes (undefined) |

> **TDZ** = Temporal Dead Zone (cannot access before declaration)

### Best Practices

```javascript
// Default to const
const API_URL = 'https://api.example.com'
const user = { name: 'John' } // Object reference is constant

// Use let when reassignment is needed
let count = 0
count++

// Avoid var (legacy)
```

### Scope Examples

```javascript
// Block scope (const, let)
if (true) {
  const x = 1
  let y = 2
}
// x, y not accessible here

// Function scope (var)
function example() {
  var z = 3
}
// z not accessible here
```

## Data Types

### Primitive Types (7)

| Type | Example | typeof |
|------|---------|--------|
| String | `'hello'`, `"world"` | `"string"` |
| Number | `42`, `3.14`, `NaN`, `Infinity` | `"number"` |
| BigInt | `9007199254740991n` | `"bigint"` |
| Boolean | `true`, `false` | `"boolean"` |
| Undefined | `undefined` | `"undefined"` |
| Null | `null` | `"object"` (quirk) |
| Symbol | `Symbol('id')` | `"symbol"` |

### Reference Types

| Type | Example | typeof |
|------|---------|--------|
| Object | `{ key: 'value' }` | `"object"` |
| Array | `[1, 2, 3]` | `"object"` |
| Function | `function() {}` | `"function"` |

### Type Checking

```javascript
// typeof for primitives
typeof 'hello'     // "string"
typeof 42          // "number"
typeof true        // "boolean"
typeof undefined   // "undefined"
typeof null        // "object" (bug)

// Array.isArray for arrays
Array.isArray([1, 2, 3])  // true
Array.isArray('hello')    // false

// instanceof for objects
[] instanceof Array       // true
{} instanceof Object      // true
```

### Type Coercion

```javascript
// Truthy values: non-zero numbers, non-empty strings, objects, arrays
// Falsy values: false, 0, -0, 0n, '', null, undefined, NaN

// String coercion
String(42)        // "42"
42 + ''           // "42"

// Number coercion
Number('42')      // 42
+'42'             // 42
parseInt('42px')  // 42
parseFloat('3.14')// 3.14

// Boolean coercion
Boolean(1)        // true
Boolean('')       // false
!!value           // Convert to boolean
```

### Comparison

```javascript
// Strict equality (recommended)
5 === 5           // true
5 === '5'         // false

// Loose equality (avoid)
5 == '5'          // true (type coercion)

// Comparing objects (by reference)
{} === {}         // false
const a = {}
const b = a
a === b           // true
```

## Functions

### Function Types

```javascript
// Function Declaration (hoisted)
function add(a, b) {
  return a + b
}

// Function Expression (not hoisted)
const subtract = function(a, b) {
  return a - b
}

// Arrow Function (no own this)
const multiply = (a, b) => a * b

// Arrow with body
const divide = (a, b) => {
  if (b === 0) throw new Error('Division by zero')
  return a / b
}
```

### Arrow Function Syntax

```javascript
// Multiple parameters
const add = (a, b) => a + b

// Single parameter (parens optional)
const double = x => x * 2

// No parameters
const greet = () => 'Hello!'

// Returning object (wrap in parens)
const createUser = name => ({ name, id: Date.now() })
```

### `this` Binding

| Function Type | `this` Behavior |
|---------------|-----------------|
| Declaration | Dynamic (depends on caller) |
| Expression | Dynamic (depends on caller) |
| Arrow | Lexical (inherits from parent) |

```javascript
const obj = {
  name: 'Object',
  regular: function() {
    console.log(this.name) // "Object"
  },
  arrow: () => {
    console.log(this.name) // undefined (window in browser)
  }
}
```

### Default Parameters

```javascript
function greet(name = 'Guest', greeting = 'Hello') {
  return `${greeting}, ${name}!`
}

greet()              // "Hello, Guest!"
greet('John')        // "Hello, John!"
greet('John', 'Hi')  // "Hi, John!"
```

### Rest Parameters

```javascript
function sum(...numbers) {
  return numbers.reduce((total, n) => total + n, 0)
}

sum(1, 2, 3, 4)  // 10
```

## Arrays

### Creating Arrays

```javascript
const arr1 = [1, 2, 3]
const arr2 = new Array(3)        // [empty Ã— 3]
const arr3 = Array.from('abc')   // ['a', 'b', 'c']
const arr4 = Array.from({ length: 3 }, (_, i) => i) // [0, 1, 2]
```

### Common Array Methods

| Method | Returns | Mutates | Description |
|--------|---------|---------|-------------|
| `map(fn)` | New array | No | Transform each element |
| `filter(fn)` | New array | No | Keep elements that pass test |
| `find(fn)` | Element/undefined | No | First element that passes |
| `findIndex(fn)` | Index/-1 | No | Index of first match |
| `some(fn)` | Boolean | No | At least one passes |
| `every(fn)` | Boolean | No | All pass |
| `reduce(fn, init)` | Any | No | Reduce to single value |
| `forEach(fn)` | undefined | No | Loop (no return) |
| `includes(val)` | Boolean | No | Check if contains |
| `indexOf(val)` | Index/-1 | No | Find index |
| `push(val)` | New length | Yes | Add to end |
| `pop()` | Removed item | Yes | Remove from end |
| `unshift(val)` | New length | Yes | Add to start |
| `shift()` | Removed item | Yes | Remove from start |
| `splice(i, n)` | Removed items | Yes | Remove/insert at index |
| `slice(start, end)` | New array | No | Extract portion |
| `sort(fn)` | Sorted array | Yes | Sort in place |
| `reverse()` | Reversed array | Yes | Reverse in place |

### Array Method Examples

```javascript
const numbers = [1, 2, 3, 4, 5]

// map - transform
numbers.map(n => n * 2)           // [2, 4, 6, 8, 10]

// filter - select
numbers.filter(n => n > 2)        // [3, 4, 5]

// find - first match
numbers.find(n => n > 2)          // 3

// reduce - accumulate
numbers.reduce((sum, n) => sum + n, 0)  // 15

// Chaining
numbers
  .filter(n => n > 2)
  .map(n => n * 2)                // [6, 8, 10]
```

### Spread Operator

```javascript
// Copy array
const copy = [...original]

// Combine arrays
const combined = [...arr1, ...arr2]

// Add items
const withNew = [...arr, newItem]

// Convert to array
const chars = [..."hello"]  // ['h', 'e', 'l', 'l', 'o']
```

## Objects

### Creating Objects

```javascript
// Object literal
const user = {
  name: 'John',
  age: 30,
  greet() {
    return `Hi, I'm ${this.name}`
  }
}

// Shorthand properties
const name = 'John'
const age = 30
const user = { name, age }  // { name: 'John', age: 30 }

// Computed properties
const key = 'status'
const obj = { [key]: 'active' }  // { status: 'active' }
```

### Accessing Properties

```javascript
const user = { name: 'John', 'full-name': 'John Doe' }

// Dot notation
user.name           // 'John'

// Bracket notation (required for special keys)
user['full-name']   // 'John Doe'

// Optional chaining
user?.address?.city // undefined (no error)
```

### Destructuring

```javascript
const user = { name: 'John', age: 30, city: 'NYC' }

// Basic
const { name, age } = user

// With alias
const { name: userName } = user

// With default
const { role = 'user' } = user

// Nested
const { address: { city } } = { address: { city: 'NYC' } }

// In function parameters
function greet({ name, age }) {
  return `${name} is ${age}`
}
```

### Spread with Objects

```javascript
// Copy object
const copy = { ...original }

// Merge objects
const merged = { ...obj1, ...obj2 }

// Override properties
const updated = { ...user, age: 31 }

// Add properties
const withRole = { ...user, role: 'admin' }
```

### Object Methods

```javascript
const obj = { a: 1, b: 2, c: 3 }

Object.keys(obj)      // ['a', 'b', 'c']
Object.values(obj)    // [1, 2, 3]
Object.entries(obj)   // [['a', 1], ['b', 2], ['c', 3]]

// Check property
'a' in obj            // true
obj.hasOwnProperty('a') // true

// Iterate
for (const [key, value] of Object.entries(obj)) {
  console.log(key, value)
}
```

## Control Flow

### Conditionals

```javascript
// if/else
if (condition) {
  // ...
} else if (otherCondition) {
  // ...
} else {
  // ...
}

// Ternary operator
const result = condition ? valueIfTrue : valueIfFalse

// Nullish coalescing
const value = input ?? 'default'  // Only for null/undefined

// Logical OR (for all falsy)
const value = input || 'default'  // For any falsy value

// Logical AND (short-circuit)
isValid && doSomething()
```

### Switch

```javascript
switch (value) {
  case 'a':
    // ...
    break
  case 'b':
  case 'c':
    // handles both b and c
    break
  default:
    // ...
}
```

### Loops

```javascript
// for
for (let i = 0; i < 5; i++) {
  console.log(i)
}

// for...of (arrays, strings, iterables)
for (const item of array) {
  console.log(item)
}

// for...in (object keys) - avoid for arrays
for (const key in object) {
  console.log(key, object[key])
}

// while
while (condition) {
  // ...
}

// do...while (runs at least once)
do {
  // ...
} while (condition)
```

### Loop Control

```javascript
// break - exit loop
for (const item of items) {
  if (item === target) break
}

// continue - skip iteration
for (const item of items) {
  if (!item) continue
  process(item)
}
```

## String Methods

### Extracting Substrings

```javascript
const str = "Hello, World!"

str.slice(0, 5)        // "Hello"
str.slice(-6)          // "World!" (from end)
str.slice(7)           // "World!" (to end)
str.substring(0, 5)    // "Hello"
```

### Searching Strings

| Method | Returns | Description |
|--------|---------|-------------|
| `includes(search)` | Boolean | Contains substring? |
| `indexOf(search)` | Index / -1 | Position of first match |
| `lastIndexOf(search)` | Index / -1 | Position of last match |
| `startsWith(search)` | Boolean | Starts with substring? |
| `endsWith(search)` | Boolean | Ends with substring? |

### Transforming Strings

```javascript
"  hello  ".trim()           // "hello"
"  hello  ".trimStart()      // "hello  "
"  hello  ".trimEnd()        // "  hello"
"hello".toUpperCase()        // "HELLO"
"HELLO".toLowerCase()        // "hello"
"hello".replace("l", "L")    // "heLlo" (first only)
"hello".replaceAll("l", "L") // "heLLo" (all)
```

### Split and Join

```javascript
"a,b,c".split(",")           // ["a", "b", "c"]
["a", "b", "c"].join("-")    // "a-b-c"
"hello".split("")            // ["h", "e", "l", "l", "o"]
```

### Padding

```javascript
"42".padStart(5, "0")        // "00042"
"42".padEnd(5, ".")          // "42..."
```

### Getting Characters

```javascript
"hello"[0]                   // "h"
"hello".charAt(0)            // "h"
"hello".at(-1)               // "o" (last character)
```

### Common Patterns

```javascript
// Clean user input
const email = "  USER@EXAMPLE.COM  ".trim().toLowerCase()

// Check file extension
filename.endsWith(".pdf")

// Format ID with leading zeros
id.padStart(6, "0")  // "000042"
```

## this Keyword

### Binding Rules

| Call Style | `this` Value |
|------------|--------------|
| `obj.method()` | `obj` (implicit binding) |
| `function()` | global / undefined (strict) |
| Arrow function | Inherits from outer scope |

### Implicit Binding

```javascript
const user = {
  name: "Alice",
  greet() {
    console.log(this.name)  // "Alice"
  }
}

user.greet()  // this = user
```

### Lost Binding (Common Pitfall)

```javascript
const user = {
  name: "Alice",
  greet() {
    console.log(this.name)
  }
}

const greet = user.greet
greet()  // this = undefined (lost binding!)
```

### Arrow Functions Preserve this

```javascript
const user = {
  name: "Alice",
  friends: ["Bob", "Charlie"],
  showFriends() {
    // Arrow function inherits 'this' from showFriends
    this.friends.forEach((friend) => {
      console.log(`${this.name} knows ${friend}`)
    })
  }
}
```

### Common Pitfall: Callbacks

```javascript
// BAD - regular function loses 'this'
setTimeout(function() {
  console.log(this.name)  // undefined
}, 1000)

// GOOD - arrow function preserves 'this'
setTimeout(() => {
  console.log(this.name)  // Works!
}, 1000)
```

### Rules of Thumb

- Use regular functions for object methods
- Use arrow functions for callbacks
- Don't use arrow functions as methods
- Look at the call site to determine `this`

## ES6+ Features

### Template Literals

```javascript
const name = 'World'
const greeting = `Hello, ${name}!`

// Multi-line
const html = `
  <div>
    <p>Paragraph</p>
  </div>
`

// Expressions
const result = `Total: ${price * quantity}`
```

### Destructuring (Quick Reference)

```javascript
// Arrays
const [first, second, ...rest] = [1, 2, 3, 4, 5]

// Objects
const { name, age = 0 } = user

// Function parameters
function fn({ a, b }, [x, y]) { }

// Swap variables
[a, b] = [b, a]
```

### Spread & Rest

```javascript
// Spread (expand)
const arr = [...arr1, ...arr2]
const obj = { ...obj1, ...obj2 }
Math.max(...numbers)

// Rest (collect)
function sum(...nums) { }
const [first, ...rest] = array
const { a, ...others } = object
```

### Short-Circuit Evaluation

```javascript
// AND (&&) - returns first falsy or last value
true && 'hello'   // 'hello'
false && 'hello'  // false

// OR (||) - returns first truthy or last value
'' || 'default'   // 'default'
'hello' || 'default' // 'hello'

// Nullish (??) - only null/undefined
null ?? 'default'    // 'default'
0 ?? 'default'       // 0
'' ?? 'default'      // ''
```

### Optional Chaining

```javascript
// Property access
user?.address?.city

// Method call
user?.getName?.()

// Array access
arr?.[0]?.name

// With nullish coalescing
const city = user?.address?.city ?? 'Unknown'
```

## Quick Reference

### Equality Comparison

```javascript
// Always use strict equality
5 === 5       // true
5 === '5'     // false
null === undefined  // false

// Object comparison (by reference)
{} === {}     // false
[] === []     // false
```

### Common Patterns

```javascript
// Default value
const value = input || 'default'
const value = input ?? 'default'  // null/undefined only

// Conditional execution
condition && doSomething()

// Ternary
const status = isActive ? 'Active' : 'Inactive'

// Object property shorthand
const user = { name, age }

// Computed property
const obj = { [dynamicKey]: value }
```

### Common Mistakes to Avoid

- Using `var` instead of `const`/`let`
- Using `==` instead of `===`
- Forgetting `return` in arrow functions with braces
- Mutating arrays/objects when you shouldn't
- Using `for...in` for arrays (use `for...of`)
- Not handling `null`/`undefined` values
- Confusing `||` with `??` for defaults
