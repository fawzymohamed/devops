# Full Practice Exam: Cloud Practitioner

This practice exam simulates the real CLF-C02 experience. It contains 25 questions spread across all four domains in their weighted proportions. Set a timer for **35 minutes** (proportional to the 90-minute, 65-question real exam) and answer each question before checking the answers.

## Exam Instructions

- **Answer every question** -- there is no penalty for guessing
- **Flag difficult questions** and come back to them
- **Watch for qualifier words** like MOST, BEST, LEAST, and NOT
- **Track your time** -- do not spend more than 2 minutes on any single question

---

## Domain 1: Cloud Concepts (6 Questions)

### Question 1

**Which of the following is a benefit of cloud computing?**

- A) You must commit to a 5-year contract
- B) You trade variable expense for upfront expense
- C) You stop guessing capacity and scale on demand
- D) You need to manage physical servers

> **Answer: C** -- "Stop guessing capacity" is one of the six benefits of cloud computing. You scale up or down based on actual demand instead of predicting future needs. Option B is backwards -- cloud trades upfront expense for variable expense.

---

### Question 2

**A software company provides email through a web browser. Users do not manage any infrastructure. Which cloud service model is this?**

- A) IaaS
- B) PaaS
- C) SaaS
- D) DaaS

> **Answer: C** -- Software as a Service (SaaS) delivers complete applications over the internet. Users access them through a browser without managing infrastructure, platforms, or code. Email services like Gmail are classic SaaS examples.

---

### Question 3

**Which AWS Well-Architected Framework pillar focuses on protecting data, systems, and assets?**

- A) Reliability
- B) Security
- C) Operational Excellence
- D) Performance Efficiency

> **Answer: B** -- The Security pillar focuses on protecting information, systems, and assets through risk assessments and mitigation strategies. It covers identity management, detective controls, infrastructure protection, data protection, and incident response.

---

### Question 4

**What is an AWS Region?**

- A) A single data center
- B) A physical location with two or more Availability Zones
- C) An edge location for caching content
- D) A virtual private network

> **Answer: B** -- An AWS Region is a geographic area containing two or more Availability Zones. Each AZ is one or more discrete data centers with independent power, networking, and cooling. Regions are isolated from each other for fault tolerance.

---

### Question 5

**How does cloud computing help organizations reduce Total Cost of Ownership (TCO)?**

- A) By requiring larger upfront investments
- B) By eliminating costs for power, cooling, facility space, and hardware maintenance
- C) By increasing the number of IT staff needed
- D) By locking organizations into single-year contracts

> **Answer: B** -- Cloud computing eliminates the need to pay for physical data center costs: building space, power, cooling, hardware purchases, and ongoing maintenance. These hidden costs of on-premises infrastructure often make cloud significantly cheaper over time.

---

### Question 6

**Which cloud deployment model keeps all resources on-premises using virtualization and cloud management tools?**

- A) Public cloud
- B) Hybrid cloud
- C) Private cloud
- D) Multi-cloud

> **Answer: C** -- A private cloud (also called on-premises cloud) uses virtualization and cloud management tools within your own data center. You get some cloud benefits (self-service, elasticity) but maintain full control and physical ownership of the infrastructure.

---

## Domain 2: Security & Compliance (7 Questions)

### Question 7

**Under the Shared Responsibility Model, which task is the CUSTOMER responsible for?**

- A) Patching the hypervisor
- B) Physical security of data centers
- C) Configuring IAM users and permissions
- D) Maintaining the global network infrastructure

> **Answer: C** -- Customers manage IAM users, groups, roles, policies, and MFA. AWS manages the hypervisor, physical security, and global network. If you configure it in the AWS console, it is your responsibility.

---

### Question 8

**Which AWS service continuously monitors for malicious activity and unauthorized behavior?**

- A) AWS CloudTrail
- B) Amazon GuardDuty
- C) AWS Config
- D) Amazon Inspector

> **Answer: B** -- Amazon GuardDuty uses machine learning and threat intelligence feeds to continuously detect threats. It analyzes CloudTrail logs, VPC Flow Logs, and DNS logs. CloudTrail records API calls. Config tracks resource changes. Inspector scans for vulnerabilities.

---

### Question 9

**What is the BEST practice for managing the AWS root user account?**

- A) Use the root user for all daily tasks
- B) Share root credentials with team leads
- C) Enable MFA on the root user and avoid using it for daily tasks
- D) Create access keys for the root user for automation

> **Answer: C** -- Enable MFA on the root user, lock away the credentials, and create IAM users for daily work. The root user has unrestricted access. Using it daily or sharing its credentials is a major security risk.

---

### Question 10

**Which IAM component should you use to grant temporary permissions to an AWS service like EC2?**

- A) IAM User
- B) IAM Group
- C) IAM Role
- D) IAM Access Key

> **Answer: C** -- IAM Roles provide temporary security credentials to AWS services. Attach a role to EC2 so it can access other services without long-term access keys. Roles are also used for cross-account access and federation.

---

### Question 11

**Which service protects against Distributed Denial of Service (DDoS) attacks?**

- A) AWS WAF
- B) AWS Shield
- C) Amazon Macie
- D) AWS Artifact

> **Answer: B** -- AWS Shield protects against DDoS attacks. Shield Standard is free and provides automatic protection for all AWS accounts. Shield Advanced adds enhanced detection, near real-time attack visibility, and 24/7 access to the AWS DDoS Response Team. WAF blocks web application exploits.

---

### Question 12

**A company needs to ensure that only encrypted connections (HTTPS) are used to access their S3 bucket. What should they configure?**

- A) A VPC security group
- B) An S3 bucket policy that denies non-SSL requests
- C) An IAM user password policy
- D) Amazon CloudFront caching

> **Answer: B** -- An S3 bucket policy with a condition that denies requests where `aws:SecureTransport` is false enforces HTTPS-only access. Security groups are for EC2 network traffic. IAM password policies control user passwords. CloudFront is a CDN.

---

### Question 13

**What does the principle of least privilege mean?**

- A) Give users the maximum permissions to avoid access issues
- B) Give users only the minimum permissions they need to do their job
- C) Create one admin account for the entire team
- D) Disable all permissions by default and never enable them

> **Answer: B** -- Least privilege means granting only the permissions required to perform a specific task -- nothing more. This reduces the blast radius if credentials are compromised. Start with no permissions and add only what is needed.

## Domain 3: Cloud Technology & Services (8 Questions)

### Question 14

**Which AWS service provides serverless compute that runs code in response to events?**

- A) Amazon EC2
- B) AWS Lambda
- C) Amazon ECS
- D) AWS Elastic Beanstalk

> **Answer: B** -- AWS Lambda runs code without provisioning servers. You upload your function, and Lambda handles scaling, patching, and infrastructure. It runs in response to events like S3 uploads, API Gateway requests, or DynamoDB changes.

---

### Question 15

**A company stores infrequently accessed data that must be available within minutes when needed. Which S3 storage class is MOST cost-effective?**

- A) S3 Standard
- B) S3 Standard-Infrequent Access (S3 Standard-IA)
- C) S3 Glacier Deep Archive
- D) S3 One Zone-IA

> **Answer: B** -- S3 Standard-IA costs less than S3 Standard for storage but charges a retrieval fee. Data is available within milliseconds. Glacier Deep Archive is cheaper but retrieval takes 12-48 hours. One Zone-IA stores data in only one AZ, which is riskier for important data.

---

### Question 16

**What is the purpose of Amazon VPC?**

- A) To host websites on AWS
- B) To create an isolated virtual network in the AWS Cloud
- C) To cache content at edge locations
- D) To manage DNS records

> **Answer: B** -- Amazon VPC (Virtual Private Cloud) lets you launch AWS resources in a logically isolated network that you define. You control IP ranges, subnets, route tables, and network gateways. It is the foundational networking service in AWS.

---

### Question 17

**Which AWS service distributes incoming traffic across multiple targets in one or more Availability Zones?**

- A) Amazon Route 53
- B) Amazon CloudFront
- C) Elastic Load Balancing
- D) AWS Direct Connect

> **Answer: C** -- Elastic Load Balancing (ELB) distributes incoming application traffic across multiple targets (EC2 instances, containers, IP addresses) across AZs. Route 53 is DNS. CloudFront is a CDN. Direct Connect is a dedicated network link.

---

### Question 18

**A developer needs a NoSQL database that provides single-digit millisecond response times at any scale. Which service should they choose?**

- A) Amazon RDS
- B) Amazon Redshift
- C) Amazon DynamoDB
- D) Amazon Aurora

> **Answer: C** -- Amazon DynamoDB is a fully managed NoSQL database that delivers single-digit millisecond performance at any scale. RDS and Aurora are relational databases. Redshift is a data warehouse for analytics.

---

### Question 19

**Which AWS service provides a content delivery network (CDN) that caches content at edge locations?**

- A) Amazon S3
- B) Amazon CloudFront
- C) Amazon Route 53
- D) AWS Global Accelerator

> **Answer: B** -- Amazon CloudFront is a CDN that caches content (web pages, images, videos, APIs) at edge locations worldwide. Users get content from the nearest edge location for low latency. S3 stores objects. Route 53 is DNS. Global Accelerator optimizes network paths.

---

### Question 20

**Which compute option automatically adds and removes EC2 instances based on demand?**

- A) Elastic Load Balancing
- B) EC2 Auto Scaling
- C) AWS Lambda
- D) Amazon Lightsail

> **Answer: B** -- EC2 Auto Scaling automatically adjusts the number of EC2 instances based on demand conditions you define. It scales out (adds instances) during high demand and scales in (removes instances) during low demand. ELB distributes traffic but does not add or remove instances.

---

### Question 21

**A company needs to connect its on-premises data center to AWS with a dedicated, private network connection. Which service should they use?**

- A) Amazon VPN
- B) AWS Direct Connect
- C) Amazon CloudFront
- D) Amazon VPC Peering

> **Answer: B** -- AWS Direct Connect establishes a dedicated, private network connection from your data center to AWS. It provides more consistent throughput and lower latency than internet-based VPN connections. VPN uses encrypted tunnels over the public internet.

---

## Domain 4: Billing, Pricing & Support (4 Questions)

### Question 22

**A company runs steady-state production workloads on EC2 and wants to save money with a long-term commitment. Which pricing model should they use?**

- A) On-Demand
- B) Spot Instances
- C) Reserved Instances or Savings Plans
- D) Dedicated Hosts

> **Answer: C** -- Reserved Instances and Savings Plans offer up to 72% savings for 1 or 3-year commitments. Steady-state production workloads are the ideal use case because usage is predictable and consistent. Spot is not suitable because instances can be interrupted.

---

### Question 23

**Which AWS tool provides a visual dashboard of your historical spending patterns and forecasts future costs?**

- A) AWS Budgets
- B) AWS Cost Explorer
- C) AWS Pricing Calculator
- D) AWS Trusted Advisor

> **Answer: B** -- AWS Cost Explorer visualizes your AWS spending over time with charts, graphs, and forecasting. It helps you analyze trends, identify cost drivers, and forecast future spending. Budgets sends alerts. Pricing Calculator estimates new architectures. Trusted Advisor gives best-practice recommendations.

---

### Question 24

**What is the benefit of using AWS Organizations with consolidated billing?**

- A) Each account gets its own separate invoice
- B) Usage across accounts is combined for potential volume discounts
- C) All services become free for member accounts
- D) Consolidated billing eliminates the need for IAM

> **Answer: B** -- Consolidated billing combines usage from all accounts in the organization. The combined usage can qualify for volume-based pricing tiers (like S3 or data transfer). You get one bill for all accounts, simplifying management. Services are not free and IAM is still needed.

---

### Question 25

**Which AWS Support plan is free and available to all AWS accounts?**

- A) Developer Support
- B) Basic Support
- C) Business Support
- D) Enterprise Support

> **Answer: B** -- Basic Support is free for all AWS accounts. It includes access to documentation, whitepapers, support forums, AWS Trusted Advisor (limited checks), and AWS Personal Health Dashboard. For technical support, you need Developer, Business, or Enterprise plans.
