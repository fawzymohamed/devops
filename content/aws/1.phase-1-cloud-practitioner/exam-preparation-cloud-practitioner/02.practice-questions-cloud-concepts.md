# Practice Questions: Cloud Concepts

## Practice Questions

### Question 1

**A startup wants to launch a web application but cannot afford to buy servers upfront. They want to pay only for compute time used. Which cloud computing benefit addresses this need?**

- A) Increase speed and agility
- B) Go global in minutes
- C) Trade upfront expense for variable expense
- D) Stop guessing capacity

> **Answer: C** -- Cloud computing lets you replace large capital expenses (buying servers) with small variable expenses (paying per hour or second of usage). The startup pays nothing upfront and only pays for what they consume.

---

### Question 2

**A company runs its application on physical servers in its own data center but uses AWS S3 for backup storage. What type of cloud deployment model is this?**

- A) Public cloud
- B) Private cloud
- C) Hybrid cloud
- D) Multi-cloud

> **Answer: C** -- A hybrid cloud connects on-premises infrastructure with public cloud resources. The company uses its own data center (private) plus AWS S3 (public cloud), making this a hybrid deployment.

---

### Question 3

**Which AWS Well-Architected Framework pillar focuses on running workloads effectively and gaining insights into operations?**

- A) Reliability
- B) Operational Excellence
- C) Performance Efficiency
- D) Security

> **Answer: B** -- Operational Excellence covers running and monitoring systems to deliver business value. This includes automating changes, responding to events, and defining standards for daily operations.

---

### Question 4

**A developer uses a managed platform to deploy code without managing servers, operating systems, or middleware. Which cloud service model is this?**

- A) IaaS
- B) PaaS
- C) SaaS
- D) On-premises

> **Answer: B** -- Platform as a Service (PaaS) handles the infrastructure, OS, and middleware. The developer focuses only on their application code and data. AWS Elastic Beanstalk is an example of PaaS.

---

### Question 5

**Which of the following are benefits of cloud computing? (Select THREE)**

- A) Benefit from massive economies of scale
- B) Eliminate the need for IT security
- C) Increase speed and agility
- D) Stop guessing capacity
- E) Remove all operational responsibility

> **Answer: A, C, D** -- These are three of the six benefits listed by AWS. Cloud computing does NOT eliminate IT security needs (customers still manage security IN the cloud) and does NOT remove all operational responsibility (you still manage your applications and data).

---

### Question 6

**What does "economies of scale" mean in the context of cloud computing?**

- A) You get a discount for being a new customer
- B) AWS passes on savings from serving hundreds of thousands of customers, resulting in lower per-unit costs
- C) Prices increase as your usage grows
- D) You must buy in bulk to get discounts

> **Answer: B** -- AWS aggregates usage from all customers. Because they buy hardware, power, and networking at massive scale, the per-unit cost is lower than what any single company could achieve. These savings are passed on as lower prices.

---

### Question 7

**A company wants to move to the cloud but must keep patient health records on its own servers due to regulations. Which deployment model fits?**

- A) Public cloud only
- B) Private cloud only
- C) Hybrid cloud
- D) Community cloud

> **Answer: C** -- Hybrid cloud allows the company to keep regulated data on-premises while moving other workloads to the public cloud. This is common in healthcare, finance, and government sectors.

---

### Question 8

**Which Well-Architected Framework pillar addresses the environmental impact of running cloud workloads?**

- A) Cost Optimization
- B) Reliability
- C) Sustainability
- D) Performance Efficiency

> **Answer: C** -- Sustainability is the newest pillar, added to help organizations minimize environmental impact. It covers efficient resource use, reducing waste, and choosing Regions with lower carbon footprints.

---

### Question 9

**A company uses Gmail for email, Salesforce for CRM, and Slack for messaging. Which cloud service model describes these tools?**

- A) IaaS
- B) PaaS
- C) SaaS
- D) FaaS

> **Answer: C** -- Software as a Service (SaaS) delivers fully managed applications over the internet. Users access the software through a browser without managing any infrastructure, platforms, or code.

---

### Question 10

**What is Total Cost of Ownership (TCO) in cloud economics?**

- A) The monthly bill from AWS
- B) The full cost of owning and operating infrastructure, including hardware, power, cooling, staff, and facilities
- C) The cost of Reserved Instances over three years
- D) The price of an AWS Support plan

> **Answer: B** -- TCO includes all costs of running infrastructure: hardware purchases, data center space, electricity, cooling, staff salaries, maintenance, and software licenses. Cloud TCO analysis compares these costs against AWS pay-as-you-go pricing to show potential savings.

## Key Takeaways

- Cloud Concepts questions test definitions, benefits, deployment models, and the Well-Architected Framework
- Watch for distractors like "eliminate all IT staff" or "remove all responsibility" -- cloud shifts responsibilities but does not eliminate them
- Know the six benefits of cloud, three service models (IaaS, PaaS, SaaS), and three deployment models (public, private, hybrid)
- The Well-Architected Framework has six pillars -- memorize their names and what each covers
