# AWS Pricing Calculator

The **AWS Pricing Calculator** is a free web tool that lets you estimate the cost of AWS services **before you deploy anything**. It helps you plan budgets, compare configurations, and build a business case for migrating to AWS.

You can access it at **calculator.aws** -- no AWS account required.

## What It Does

The Pricing Calculator lets you:

- **Add AWS services** and configure them (instance type, storage size, data transfer, Region)
- **See monthly and annual cost estimates** based on your configuration
- **Compare pricing models** (On-Demand vs. Reserved vs. Savings Plans) side by side
- **Organize services into groups** for multi-tier applications
- **Share estimates** with your team via a unique URL
- **Export estimates** as CSV or PDF for documentation

### Step 1: Add a Service

Search for any AWS service (EC2, S3, RDS, Lambda, etc.) and add it to your estimate.

### Step 2: Configure the Service

Set the parameters that affect pricing:

| Service | Key Configuration Options |
|---------|--------------------------|
| **EC2** | Instance type, Region, operating system, number of instances, hours per month |
| **S3** | Storage amount (GB), number of requests, data transfer out |
| **RDS** | Instance type, database engine, storage size, Multi-AZ deployment |
| **Lambda** | Number of requests, memory allocated, execution duration |

### Step 3: Review and Adjust

The calculator shows your **estimated monthly cost** for each service and the total. You can adjust configurations, try different Regions (prices vary by Region), or compare On-Demand vs. Reserved pricing.

### Step 4: Share or Export

Generate a **shareable link** so teammates can view and modify the estimate. You can also export it as a CSV or PDF for proposals and budget approvals.

## Estimate Groups

For complex applications, organize services into **groups** that represent different tiers or environments.

**Example:** A web application estimate might have:

| Group | Services Included |
|-------|------------------|
| **Web Tier** | EC2 instances, Elastic Load Balancer, CloudFront |
| **Application Tier** | Lambda functions, API Gateway |
| **Database Tier** | RDS instance, ElastiCache |
| **Storage** | S3 buckets, EBS volumes |

Each group shows its subtotal, and the calculator shows the overall total for the entire architecture.

## Pricing Calculator vs. Other Cost Tools

| Tool | When to Use |
|------|-------------|
| **Pricing Calculator** | **Before** deployment -- estimate future costs |
| **Cost Explorer** | **After** deployment -- analyze actual spending |
| **AWS Budgets** | **Ongoing** -- set alerts when spending exceeds thresholds |
| **Trusted Advisor** | **Ongoing** -- find opportunities to reduce costs |

The Pricing Calculator is the **starting point** in your cost management journey. Once you deploy, switch to Cost Explorer and Budgets to track real spending.

## Best Practices

- **Estimate before you build** -- always run a cost estimate before launching a new project or migrating a workload
- **Try multiple Regions** -- prices vary by Region, and choosing a less expensive Region can save money
- **Compare pricing models** -- use the calculator to see how much you save with Reserved Instances or Savings Plans vs. On-Demand
- **Include data transfer** -- do not forget outbound data transfer costs, which are often overlooked
- **Update estimates regularly** -- as your architecture changes, revisit and adjust your estimates

## Key Takeaways

- The AWS Pricing Calculator is a **free, public web tool** at calculator.aws -- no AWS account needed
- Use it to estimate costs **before** you deploy services on AWS
- Configure services with specific instance types, Regions, and usage patterns for accurate estimates
- Organize multi-service architectures into **groups** for clarity
- Share estimates via URL or export as CSV/PDF
- Use the Pricing Calculator for planning; switch to Cost Explorer and Budgets for tracking real costs
