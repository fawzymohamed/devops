# Machine Learning & AI Services Overview

AWS offers two types of AI/ML services: **pre-built AI services** that work out of the box and **custom ML platforms** for building your own models. You do not need machine learning expertise to use most of these services -- they are accessible through simple API calls.

## Two Approaches to AI on AWS

| Approach | ML Expertise Needed? | Services |
|----------|---------------------|----------|
| **Pre-built AI services** | No -- just call an API | Rekognition, Comprehend, Polly, Lex, Translate, Textract, Forecast |
| **Custom ML platform** | Yes -- build your own models | SageMaker, Bedrock |

## Pre-Built AI Services

These services solve common AI tasks without requiring you to build or train models. You send data to an API and get results back.

### Amazon Rekognition

Analyzes **images and videos** to detect objects, faces, text, scenes, and activities.

- Identify products on store shelves
- Detect unsafe content in user-uploaded images
- Verify identity by comparing faces

### Amazon Comprehend

Performs **natural language processing (NLP)** on text to extract meaning.

- Detect sentiment (positive, negative, neutral) in customer reviews
- Extract key phrases and entities (names, dates, locations) from documents
- Identify the language of a text

### Amazon Polly

Converts **text into lifelike speech** in dozens of languages and voices.

- Add voice output to applications for accessibility
- Generate audio versions of articles and reports
- Create voice prompts for phone systems

### Amazon Lex

Builds **conversational chatbots** that understand natural language. Lex powers the same technology behind Alexa.

- Build customer service chatbots for websites
- Create voice-enabled IVR (phone menu) systems
- Integrate with Lambda to execute business logic based on user intent

### Amazon Translate

Performs **real-time language translation** across 75+ languages.

- Translate user-generated content on global platforms
- Localize applications and documentation
- Enable multilingual customer support

### Amazon Textract

Extracts **text, handwriting, and structured data** from scanned documents.

- Pull data from invoices, receipts, and tax forms automatically
- Extract table data from scanned PDFs
- Process handwritten applications and forms

### Amazon Forecast

Generates **time-series forecasts** using machine learning. You provide historical data and Forecast predicts future values.

- Predict product demand for inventory planning
- Forecast website traffic for capacity planning
- Estimate revenue for financial planning

## Custom ML Platforms

### Amazon SageMaker

The **end-to-end platform** for building, training, and deploying custom ML models. SageMaker handles the heavy infrastructure work so data scientists can focus on their models.

| Capability | What It Does |
|------------|-------------|
| **Build** | Jupyter notebooks, built-in algorithms, data labeling tools |
| **Train** | Managed training infrastructure, automatic model tuning |
| **Deploy** | One-click deployment, auto-scaling endpoints, A/B testing |

Use SageMaker when pre-built services do not fit your needs -- for example, predicting customer churn based on your specific business data.

### Amazon Bedrock

Access **foundation models** (large language models) from Amazon and third-party providers to build generative AI applications. Bedrock lets you use powerful AI models without training them yourself.

- Build AI-powered content generators
- Create intelligent search and Q&A systems
- Customize models with your own data using fine-tuning

## Choosing the Right Service

| Business Problem | AWS Service |
|-----------------|-------------|
| Analyze images or videos | Rekognition |
| Understand text sentiment or entities | Comprehend |
| Convert text to spoken audio | Polly |
| Build a chatbot | Lex |
| Translate between languages | Translate |
| Extract data from documents | Textract |
| Predict future values from past data | Forecast |
| Build a custom ML model | SageMaker |
| Build with large language models | Bedrock |

## Key Takeaways

- AWS pre-built AI services need **no ML expertise** -- just call an API with your data
- **Rekognition** analyzes images/video; **Comprehend** analyzes text meaning; **Textract** extracts data from documents
- **Polly** does text-to-speech; **Lex** builds chatbots; **Translate** handles language translation
- **SageMaker** is for building **custom ML models** when pre-built services do not fit your use case
- **Bedrock** gives you access to foundation models for generative AI without training from scratch
- For the Cloud Practitioner exam, know **what each service does** and **when to use it** -- not how to build models
